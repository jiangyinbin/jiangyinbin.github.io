Twitter个性化分享调研

  • SDK
  • Twitter
  • 分享
  • 原创

通过接入Twitter的SDK来达到高转化率的个性化社交推广！

由于调用TwitterSDK需要在开发者后台创建App等，较为麻烦，直接使用我配置好的KEY就行。

SECRET：FQNS0eBbM8J5DDjDglLCwOvAUUesYhN5uFxdxjKLgk

1.引入Twitter项目包

build.gradle添加 

dependencies {
    compile('com.twitter.sdk.android:twitter:1.9.1@aar') {
        transitive = true;
    }
}

安装官方文档的配置会发现下载不下来，原因是twitter没有把代码放到自己的项目中，而是放到自己的网站上托管了…

加入Twitter的Maven库就搞定了

repositories {
    maven{ url 'https://maven.fabric.io/repo'}
}

分享成功和失败的广播回调 

<receiver
  android:name=".MyResultReceiver"
  android:exported="false">
      <intent-filter>
        <action android:name="com.twitter.sdk.android.tweetcomposer.UPLOAD_SUCCESS"/>
        <action android:name="com.twitter.sdk.android.tweetcomposer.UPLOAD_FAILURE"/>
      </intent-filter>
</receiver>

2.AppLinks

在Twitter后台配置应用连接，然后使用此连接分享到Twitter即可直接打开自己的App

3.App Installs

分享到此类型博客到Twitter后点击install及可下载安装，很适合做App推广

4.TwitterCard

卡片式分享功能可现实多张图片缩略图，标题，作者和文字描述，以及大相册链接等。

5.分享话题

可以使用一个#号来发布话题，如： #Android6.0大法好

Android主题、皮肤切换，支持在线安装，可直接替换整个布局
